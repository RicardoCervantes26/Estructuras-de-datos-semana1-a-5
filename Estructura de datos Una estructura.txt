Estructura de datos: Una estructura de datos es una forma que se puedan almacenar datos y realizar operaciones de manera eficiente y esta estructura de datos tiene y las computadoras almacenan los datos en la memoria RAM 
SE DIVIDEN EN ESTATITICAS Y DINAMICAS:
LAS ESTATICAS: no se puede cambiar el tiempo de copilacion como arreglos y matrizes
LAS DINAMICAS: se puede cambiar el tiempo de copilacion y estos mismos se dividen en :
 -lineales los elementos se organizan secuencialmente como       Pilas y Colas
 - no lineales los elementos no se organizan de mnera secuencial
 como arboles graphos y tablas hash.
